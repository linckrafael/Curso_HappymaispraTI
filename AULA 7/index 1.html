GIT E GITHUB

O sistema Git é usado para controlar o histórico de alterações de arquivos, muito usado por desenvolvedores. Ele permite mais segurança, fluxo e desempenho. O GitHub é uma plataforma onde pessoas, principalmente desenvolvedores, colocam seus projetos, seus códigos, seus arquivos e podem ou não compartilhar com outros desenvolvedores. É quase como uma rede social para desenvolvedores e uma nuvem, porém um pouco mais complexo, envolvendo questões de segurança principalmente. Fontes: https://blog.betrybe.com/git/
